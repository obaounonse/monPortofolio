Cette application To-Do List est une solution web interactive permettant aux utilisateurs de gérer efficacement leurs tâches quotidiennes. Elle offre une interface intuitive et moderne avec plusieurs fonctionnalités avancées

Technologies utilisées : HTML, CSS et JavaScript.

Voir le Projet

Fonctionnalités

  • ✅ Ajout de tâches avec description, priorité et date limite
  • ✅ Marquage des tâches comme terminées
  • ✅ Filtrage des tâches (toutes, en cours, terminées)
  • ✅ Mode sombre activable
  • ✅ Affichage de l'heure d'ajout et rappels pour les échéances
  • ✅ Progression des tâches via une barre de suivi
  • ✅ Stockage local des tâches pour éviter toute perte de données
